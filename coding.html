Coding

I completed CS50x: CS50's Introduction to Computer Science, and currently taking the next course, CS50's Web Programming with Python and Javascript. I also spent a year learning to code through freecodecamp, where I earned their Full Stack Certification. On my personal webpage I have included most of my coding accomplishments, including my projects through freecodecamp, my final project for CS50x (the course's projects are at submit50/raphaeluziel), along with a link to my old physics class website, that started my coding journey.
